# simpler-sor - A Simpler Approach SDK

This SDK is a publishable library designed to facilitate building CRUD systems using the "A Simpler Approach" pattern.

## Overview

The "A Simpler Approach" pattern leverages serverless technologies (like AWS DynamoDB and EventBridge) to create scalable, maintainable, and cost-effective applications with minimal boilerplate. This SDK abstracts away common infrastructure and pattern implementation details.
